3. Attribute (href, src, alt)

Attribute erweitern HTML-Tags um zusätzliche Informationen. Links benötigen ein Ziel, Bilder eine Quelle.

<a href="https://google.com">Link</a>
<img src="bild.png" alt="Beschreibung">

3. Attribute

Attribute sind wie kleine Zusatznotizen, die man an die Tags heftet, um ihnen mehr Details zu geben. Ein Tag allein ist oft zu ungenau. Nehmen wir das Bild-Tag (img): Der Browser weiß zwar jetzt, dass dort ein Bild hin soll, aber er weiß nicht, welches! Hier kommt das Attribut "src" (source, also Quelle) ins Spiel. Du schreibst src="hund.jpg" in den Tag, und schon weiß der Browser, welche Datei er laden muss. Ein weiteres super wichtiges Attribut für Bilder ist "alt". Das ist ein kurzer Text, der beschreibt, was auf dem Bild zu sehen ist. Wenn das Internet mal langsam ist und das Bild nicht lädt, wird dieser Text angezeigt. Außerdem ist es wichtig für blinde Menschen, damit ihr Computer ihnen vorlesen kann, was dort zu sehen wäre. Links brauchen das Attribut "href", damit sie wissen, welche Internetseite sie öffnen sollen, wenn man draufklickt. Es gibt auch Attribute für Namen (Klassen und IDs), damit man später mit CSS sagen kann: "Nur dieses eine Element mit diesem Namen soll rot werden". Man kann sich Attribute wie die Ausstattung bei einem Auto vorstellen: Das Auto ist das Tag, aber ob es eine Klimaanlage, eine bestimmte Farbe oder ein Navi hat, bestimmen die Attribute.
